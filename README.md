# Kretek

Uma landing page meme e dramática para responder à pergunta mais importante da noite: **fumaria um kretek?**

## Funcionalidades

- Fluxo em etapas: convite, escolha de data/horário e confirmação.
- Botão `Não` fugitivo no desktop e no mobile.
- Calendário e seletor de horários customizados.
- Animações com `framer-motion`.
- Visual responsivo com imagem animada do kretek também no mobile.
- Link de confirmação no WhatsApp com mensagem preenchida automaticamente.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Rodando Localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
